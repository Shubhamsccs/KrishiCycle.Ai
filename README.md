# KrishiCycle.Ai 🌱

A web application built during a hackathon to promote sustainable agriculture. The project focuses on the concept of "Waste to Wealth" by providing guidance and tools to turn bio-waste into valuable organic fertilizer.

## 🚀 The Challenge & Solution
* **Problem:** Massive daily bio-waste, reliance on harmful artificial fertilizers, and a knowledge gap in effective composting.
* **Solution:** A platform providing step-by-step guidance, custom compost recipes, and a marketplace concept for organic materials.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Concept:** UI/UX for sustainable agricultural solutions

## 💡 What I Learned
Participating in this hackathon taught me how to structure a project under time constraints, collaborate on real-world problem statements, and build user-friendly interfaces to convey complex information clearly.

## ⚙️ How to Run
1. Clone this repository.
2. Open the folder and launch `index.html` (or the main html file) in your browser.
